Nexus-server installation in AWS EC2 Instance (Amazon linux-machine):
----------------------------------------------
Pre-requisite softwares
----------------------
Java is the pre-requisite software for nexus server

java -version

17

Hardware Requirements:
----------------------
Nexus server requires at least 2GB of RAM

free -h

sudo su -

cd /opt/

yum install tar wget -yum

wget https://download.sonatype.com/nexus/3/nexus-3.73.0-12-unix.tar.gz

tar -zvxf nexus-3.73.0-12-unix.tar.gz

ls

mv /opt/nexus-3.73.0-12 /opt/nexus

// As a good practice, nexus is not advised to run nexus as a root user, so create a new user called "nexus" and grant sudo access to manage nexus services as follows.

useradd nexus

// Give the sudo access to nexus user

vi sudo
## Allow root to run any commands anywhere
root ALL=(ALL)   ALL (In this section please add below line)

nexus ALL=(ALL)  NOPASSWD: ALL

// Change the owner and group permissions to /opt/nexus and /opt/sonatype-work directories.

chown -R nexus:nexus /opt/nexus
chown -R nexus /opt/sonatype-work

chmod -R 775 /opt/nexus
chmod -R 775 /opt/sonatype-work

// Open /opt/nexus/bin/nexus.rc file, uncomment run_as_user parameter and set as nexus user

vi /opt/nexus/bin/nexus.rc
rub_as_user="nexus"

// Add nexus as a service at boot time

ln -s /opt/nexus/bin/nexus /etc/init.d/nexus

// switch as a nexus user and start the nexus service as follows.

su - nexus

// Enable the nexus services

sudo systemctl enable nexus
sudo systemctl start nexus
sudo systemctl status nexus

By default nexus server is running on port number "8081"

Make sure you will add port 8081 in security groups.

http://<public-ip>:8081/

Default credentials:

username:admin
password:admin123
